# 🛡️ Development Guardrails & Permissions

## 1. Naming Conventions (SACRED)
- **Functions, Methods, Variables**: MUST use `snake_case`.
- **Classes, Interfaces, Types, Components**: MUST use `PascalCase`.
- **Package Manager**: ALWAYS use `Bun` for javascript and typescript. ALWAYS use `UV` for python.
